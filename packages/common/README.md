## 简介

Hocuspocus 是为 [Tiptap](https://github.com/ueberdosis/tiptap) 提供的一个有主见的协作编辑后端 —— 基于 [Y.js](https://github.com/yjs/yjs)，这是一个具有强大共享数据抽象能力的 CRDT 框架。
